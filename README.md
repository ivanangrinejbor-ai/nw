### Что нового в этой версии

- **Поиск пути (A\*)**: сетка, обход препятствий, сглаживание, два режима — «точно в координату» и «до касания», авто-перестроение маршрута
- **Переходы сцен**: затемнение (Fade Out) и появление (Fade In), настраиваемая длительность 0.1–40 сек, кнопка в списке сцен
- **Глобальная сцена**: исправлено зависание скриптов при переходе между сценами
- **Экспорт с паролем**: AES-256-GCM шифрование проекта при экспорте, импорт с проверкой пароля
- **Сборка APK**: кнопка «Build APK» в опциях проекта, авто-шаблон через `./gradlew copyTemplateApk`
- **Firebase**: включён в APK-темплейте, интернет-проверка, без чёрного экрана при ошибке
- **Серый дизайн**: новый цвет фона `#3C3C3C`

### Основные возможности

| Категория | Что есть |
|-----------|----------|
| **3D** | Сцены, физика, PBR, постобработка, ECS-редактор с иерархией |
| **2D** | Fast2D с ECS, сотни объектов, физика Box2D |
| **AI** | ONNX-модели, Gemini API, PocketTensor, Code Analysis, AI-ассистент |
| **Python 3.12** | Полный Python, pip, изолированное окружение (ARM64) |
| **Qemu** | Эмуляция x86_64 через блоки (ARM64) |
| **LunoScript** | Свой язык (Kotlin-подобный), работа с Java-объектами в рантайме |
| **Git** | Встроенный Git, совместная разработка проектов |
| **Firebase** | Чтение/запись/удаление, защита от отсутствия интернета |
| **Интернет** | HTTP-запросы, WebSocket, локальный сервер |
| **Плагины** | Кастомизация интерфейса через плагины |
| **Библиотеки** | Создание своих блоков и формул |
| **Файлы** | Изолированное хранилище на проект, операции с файлами |
| **UI** | Горячие клавиши, ПК-режим, переключение стиля скролла |
| **Анализ** | Статический анализ кода, подсветка ошибок в реальном времени |

### Быстрый старт

```bash
git clone https://github.com/Danveyd/NeoCatroid.git
cd NeoCatroid
# Открыть в Android Studio, дождаться синхронизации Gradle
```

Требования: **JDK 11**, Android Studio последней стабильной версии.

### Сборка APK-темплейта

```bash
./gradlew copyTemplateApk   # собирает apktemplate flavour и копирует в assets/
```
