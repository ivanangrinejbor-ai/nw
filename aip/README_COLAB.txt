================================================================================
  ИНСТРУКЦИЯ: ОБУЧЕНИЕ LSTM-НЕЙРОСЕТИ В GOOGLE COLAB (GPU)
================================================================================

Требуется: Google аккаунт, файл colab_pack.zip + train_colab.ipynb из aip/

================================================================================
  Шаг 1: ПОДГОТОВКА (на этом ПК)
================================================================================

Запусти aip\prepare_colab.bat
  -> создаст colab_pack.zip (внутри projects.json = 342 проекта)

================================================================================
  Шаг 2: ЗАГРУЗКА В GOOGLE DRIVE
================================================================================

1. https://drive.google.com
2. Загрузи: colab_pack.zip + train_colab.ipynb

================================================================================
  Шаг 3: COLAB
================================================================================

1. https://colab.research.google.com
2. Файл -> Загрузить ноутбук -> train_colab.ipynb
3. Runtime -> Change runtime type -> T4 GPU
4. Выполни ячейки по порядку:
   Ячейка 0: mount Drive + распаковать zip:

       from google.colab import drive
       drive.mount('/content/drive')
       !cp /content/drive/MyDrive/colab_pack.zip /content/
       !cd /content && unzip -o colab_pack.zip
       !ls /content/training_data/

   Ячейка 1: импорт TF, проверка GPU
   Ячейка 2: настройки (WINDOW=500, EPOCHS=15, ...)
   Ячейка 3: загрузка projects.json (342 проекта)
   Ячейка 4: построение словаря
   Ячейка 5: генерация training pairs
   Ячейка 6: ОБУЧЕНИЕ LSTM на GPU (~2 мин/эпоху)
   Ячейка 7: конвертация в TFLite + сохранение
   Ячейка 8: скачать model.tflite + vocab.json + model_metadata.json

================================================================================
  Шаг 4: ДЕПЛОЙ В APK
================================================================================

Скачанные файлы скопируй в catroid\src\main\assets\
Или запусти aip\deploy.bat (если файлы в aip\model\)

================================================================================
  ПРИМЕЧАНИЯ
================================================================================

- Бесплатный Colab даёт ~4 часа GPU в день, хватит
- Если ошибка памяти -> уменьши BATCH_SIZE до 64
- Если projects.json не найден -> ноутбук предложит загрузить вручную
- После копирования в assets можно собирать APK
- RTX 2050 на компе тоже можно, но нужен WSL2 или DirectML plugin
