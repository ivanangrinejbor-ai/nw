===============================================================================
  ИНСТРУКЦИЯ: ОБУЧЕНИЕ В GOOGLE COLAB (GPU)
===============================================================================

Требуется: Google аккаунт, файл colab_pack.zip из aip/

===============================================================================
  ВЫБОР МОДЕЛИ
===============================================================================

1. Transformer (рекомендуется):  train_colab_transformer.ipynb
   - Окно 5000 токенов, декодерный Transformer
   - Требует ~12GB GPU VRAM (T4 подходит)
   - Сохраняет: transformer_model.pt / transformer_model.ptl

2. LSTM (легковесный):  train_colab.ipynb
   - Окно 500 токенов, модель ~2MB в TFLite
   - Работает на любом GPU
   - Сохраняет: model.tflite

===============================================================================
  Шаг 1: ПОДГОТОВКА (на этом ПК)
===============================================================================

Запусти aip\prepare_colab.bat
  -> создаст colab_pack.zip (внутри projects.json с проектами)

===============================================================================
  Шаг 2: ЗАГРУЗКА В GOOGLE DRIVE
===============================================================================

1. https://drive.google.com
2. Загрузи: colab_pack.zip

===============================================================================
  Шаг 3: COLAB
===============================================================================

1. https://colab.research.google.com
2. Файл -> Загрузить ноутбук -> train_colab_transformer.ipynb
3. Runtime -> Change runtime type -> T4 GPU
4. Выполни ячейки по порядку:
   Ячейка 0: импорт библиотек
   Ячейка 1: настройки (WINDOW=5000, EPOCHS=20, ...)
   Ячейка 2: загрузить projects.json
   Ячейка 3: построение словаря со структурными токенами
   Ячейка 4: сборка датасета (5000 токенов контекст)
   Ячейка 5: создание Transformer модели
   Ячейка 6: ОБУЧЕНИЕ (~3-5 мин/эпоху на T4)
   Ячейка 7: сохранение + TorchScript
   Ячейка 8: скачать результаты

===============================================================================
  Шаг 4: ДЕПЛОЙ В APK
===============================================================================

Transformer: скопируй transformer_model.onnx + vocab.json + model_metadata.json
LSTM:        скопируй model.tflite + vocab.json + model_metadata.json

В catroid\src\main\assets\
Или запусти aip\deploy.bat (если файлы в aip\model\)

Для трансформера используй блоки LoadNN/PredictNN в редакторе —
ONNX Runtime уже встроен в приложение.
Для интеграции в AI Assistant (авто-подсказки) нужно будет дописать
ONNX-загрузчик в NeuralSuggestionEngine (пока использует только TFLite).

===============================================================================
  ПРИМЕЧАНИЯ
===============================================================================

- Бесплатный Colab даёт ~4 часа GPU в день, хватит
- Если ошибка памяти -> уменьши BATCH_SIZE до 4 или WINDOW до 2048
- Если projects.json не найден -> ноутбук предложит загрузить вручную
- После копирования в assets можно собирать APK
- Локально: RTX 2050 тоже подойдёт, batch_size=4
